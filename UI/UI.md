# UI - Модификация для создания графических интерфейсов (Tkinter)

## Описание
Модификация UI предоставляет блоки для создания графических интерфейсов с использованием встроенной библиотеки Tkinter. Это простая и легкая альтернатива для создания оконных приложений.

## Необходимые библиотеки

Эта модификация использует встроенную библиотеку Python `tkinter`, которая обычно уже установлена вместе с Python. Поэтому **не требует установки дополнительных библиотек**.

## Как добавить библиотеки в Setup

Данная модификация не требует добавления библиотек в `requirements.txt`, так как использует встроенные возможности Python.

Если вы хотите посмотреть раздел Setup:
1. Откройте редактор TurtCD
2. Нажмите на кнопку **Настройки** (иконка шестеренки) в правом верхнем углу
3. В открывшемся окне найдите раздел **Setup**

## Основные возможности

- Создание окон приложений
- Работа с кнопками, метками, полями ввода
- Создание текстовых областей
- Работа с диалоговыми окнами
- Работа с файлами
- Создание меню

## Пример использования

1. Используйте блок **"Инициализировать tkinter"** для импорта модулей
2. Создайте окно с помощью блока **"Создать окно"**
3. Добавляйте элементы интерфейса (кнопки, метки, поля ввода)
4. Настраивайте обработчики событий
5. Запустите главный цикл с помощью блока **"Запустить главный цикл"**

## Отличия от FreshUI

- UI использует Tkinter (встроенная библиотека, проще)
- FreshUI использует PySide6 (более современная и функциональная)
