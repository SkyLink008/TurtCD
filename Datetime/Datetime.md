# Datetime - Модификация для работы с датой и временем

## Описание
Модификация Datetime предоставляет блоки для работы с датой и временем в Python. Эта модификация использует встроенную библиотеку `datetime`, которая не требует дополнительной установки.

## Необходимые библиотеки

Эта модификация использует встроенную библиотеку Python `datetime`, поэтому **не требует установки дополнительных библиотек**.

## Как добавить библиотеки в Setup

Данная модификация не требует добавления библиотек в `requirements.txt`, так как использует встроенные возможности Python.

Если вы хотите посмотреть раздел Setup:
1. Откройте редактор TurtCD
2. Нажмите на кнопку **Настройки** (иконка шестеренки) в правом верхнем углу
3. В открывшемся окне найдите раздел **Setup**

## Основные возможности

- Получение текущей даты и времени
- Форматирование даты и времени
- Работа с временными метками
- Вычисление разницы между датами
- Преобразование строк в объекты даты

## Пример использования

1. Используйте блок **"Инициализировать datetime"** для импорта модуля
2. Используйте блоки получения текущей даты/времени
3. Форматируйте дату с помощью блоков форматирования
4. Выполняйте вычисления с датами
