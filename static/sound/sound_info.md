# Список звуковых файлов

Эта папка содержит звуковые файлы для интерфейса TurtCD.

## Необходимые звуковые файлы

### Звуки кнопок

1. **button_click.mp3** или **button_click.wav**
   - Звук при нажатии на обычные кнопки
   - Используется для всех кнопок интерфейса (кроме специальных случаев)

2. **button_click_primary.mp3** или **button_click_primary.wav**
   - Звук при нажатии на основные кнопки (сохранить, скомпилировать, экспорт)
   - Более выразительный звук для важных действий

3. **button_click_ghost.mp3** или **button_click_ghost.wav**
   - Звук при нажатии на вторичные кнопки (ghost buttons)
   - Более тихий звук для менее важных действий

4. **button_toggle.mp3** или **button_toggle.wav**
   - Звук при переключении переключателей (toggle switches)
   - Используется для включения/выключения модификаций, настроек

### Звуки уведомлений

5. **notification_success.mp3** или **notification_success.wav**
   - Звук при появлении уведомления об успехе
   - Позитивный, приятный звук

6. **notification_error.mp3** или **notification_error.wav**
   - Звук при появлении уведомления об ошибке
   - Предупреждающий звук

7. **notification_info.mp3** или **notification_info.wav**
   - Звук при появлении информационного уведомления
   - Нейтральный звук

8. **notification_warning.mp3** или **notification_warning.wav**
   - Звук при появлении предупреждающего уведомления
   - Внимательный звук

## Формат файлов

- Поддерживаемые форматы: `.mp3`, `.wav`, `.ogg`
- Рекомендуемый формат: `.mp3` (лучшая совместимость)
- Рекомендуемая длительность: 0.1-0.5 секунды для кнопок, 0.3-1 секунда для уведомлений
- Рекомендуемый битрейт: 128 kbps для MP3

## Поведение

- Если звуковой файл отсутствует, звук не будет воспроизведен (без ошибок)
- Звуки воспроизводятся асинхронно и не блокируют интерфейс
- Громкость звуков можно регулировать через системные настройки

## Настройка громкости

Громкость звуков настраивается в файле `volume.json` в этой же папке.

### Параметры файла volume.json:

```json
{
  "volume": 100
}
```

- `volume` - громкость звуков в процентах (от 0 до 100)
- По умолчанию: 100 (максимальная громкость)
- Значение 0 отключает звуки
- Значения автоматически ограничиваются диапазоном 0-100

### Изменение громкости:

1. Откройте файл `static/sound/volume.json`
2. Измените значение `volume` (например, `50` для 50% громкости)
3. Сохраните файл
4. Перезагрузите страницу редактора для применения изменений

## Отключение звуков

Звуки можно отключить:
- Установив `volume: 0` в файле `volume.json`
- Через настройки браузера или системные настройки звука
